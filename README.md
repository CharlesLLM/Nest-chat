# Nest Chat

Web application for real-time chat using NestJs and Vue.js

## Installation and setup

1. Install the dependencies:

```bash
make install
```

2. Start the db container:

```bash
make docker-up
```

3. Create and migrate the database:

```bash
make db
```

4. Start the application:

```bash
make back-start
```

5. Open another terminal and run the front-end application:

```bash
make front-start
```
